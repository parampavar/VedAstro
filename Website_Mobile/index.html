Quick Guide

This is a web app to help you use vedic astrology. It is designed to work like an app running in the cloud.

How To Use?

STEP 1
Choose or add person's horoscope
STEP 2
Choose calculator to use
STEP 3
Click calculate for predictions. Easy!

What is a calculator?

Tutorials

Each calculator shows an aspect of astrology. For example Match is a calculator for checking marriage or friendship compatibility. Life Predictor can help predict major life events decades ahead and using Good Time Finder you can find the perfect time for important occasions like marriage, job interviews or starting a project. Explore each calculator to get a deep understanding of Vedic Astrology.

Unexpected errors?

Press CTRL + SHIFT + R. This will clear the old copy (cache), and solve 90% of the errors you face. VedAstro gets new updates almost everyday!

So it's important that you're using the latest version. The browser caches the old version, which causes errors as it fails to work with the updated API Server. For Mac OSX users, try COMMAND + SHIFT + R in Chrome.

Who is this for?

Both an average person and expert astrologer can find VedAstro useful. In every calculator there is an "Easy" and "Advanced" version of the results.

These calculators a constantly being improved. The ideal goal of this project is to make all tools easily usable by anybody. So if you find any of the calculator too difficult, just leave a note and it might be better the next time you visit.

Must I login to use?

No. VedAstro was designed with ease and speed in mind. So all tools are available for use without login. But if you don't login, we can't save the data you add. For example, if you add a Person profile it will only be available the next time you visit if you login first.

Else it will be deleted when you close & clear cache. To login is very simple using Google & Facebook. So you don't have to waste your time creating a new account with boring passwords to remember. We strongly believe in user privacy. So we don't collect or connect to any data from your Google or Facebook account, it is only to authenticate your access. You can verify this in the open source code.

Info For Coders

Under the hood, VedAstro is a hybrid Blazor WebAssembly + JS web application. The cloud method of using VedAstro is the recommended way for most users. But if you're feeling adventurous, try out VedAstro in Windows & Open API.

These 4 parts come together to make the final service you see. You can incorporate individual parts of VedAstro into your own projects as well. You can find the Windows exe build in our github repo. If you need help with our API or Code, just drop us a mail.

What is Numerology?

To put things in a nutshell, every man or woman is represented by a Number (since he is born on a particular date, month and year) and also defined by letters pertaining to their names.

According to these sastras the man is a Yantra and his name is a Mantra. Favourable results can be obtained only if both agree with each other and any disagreement will be harmful to them. This is the basic foundation of this science.

Source of Numerology?

Mantra Sastra helps us to understand the latent powers of nature, and we learn to command them through sound vibrations. Finding out the forms of those invisible powers and then using them is the aim of Tantra Sastra.

Only these two mutually related sastras can help us to understand nature and live in accordance with it. These two sastras have been kept secret to this day lest it should fall in the hands of the unscrupulous. These are still sustained only under a Guru-Disciple set-up.
